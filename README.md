# FullStack-Project
Full-Stack project, included both client and server side.
Client-Side: React.
Server-Side: NodeJS & Express.
Data-Base: MongoDB.
This project is about an online course website, if somebody wants to watch the content of the course
he should first register and then activate his account by entering the link sent to his registration email.
One can update his password and name.
There is also 'About' and 'Recommendation' pages.
The website is deployed using heroku deployment.
Website Link: http://theowebrotem.herokuapp.com/
